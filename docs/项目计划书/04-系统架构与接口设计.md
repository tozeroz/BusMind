# 四、系统架构与接口设计

## 八、系统总体架构

本项目采用前后端分离架构，整体架构如下：

```text
用户浏览器
   ↓
Vue3 前端页面
   ↓ Axios 请求
FastAPI 后端服务
   ↓
业务服务层
   ├── 用户服务
   ├── 公交查询服务
   ├── ETA 计算服务
   ├── 客载量预测服务
   ├── 舒适推荐服务
   └── AI 助手服务
   ↓
数据访问层 SQLAlchemy
   ↓
MySQL 数据库

FastAPI 后端
   ↓
Python 算法模块
   ├── 数据模拟
   ├── ETA 计算
   ├── 客载量预测
   └── 推荐评分

FastAPI 后端
   ↓
大模型 API
   ↓
自然语言出行建议
```

## 九、技术选型

| 模块       | 技术选型                         | 说明                                       |
| ---------- | -------------------------------- | ------------------------------------------ |
| 前端框架   | Vue3                             | 构建用户交互页面                           |
| UI 组件    | Element Plus                     | 快速开发表单、表格、卡片等页面组件         |
| 地图展示   | 高德地图 API / 百度地图 API      | 展示线路、站点、车辆位置                   |
| 图表展示   | ECharts                          | 展示拥挤趋势、客流统计等图表               |
| 后端框架   | FastAPI                          | 提供 RESTful API，自动生成接口文档         |
| 数据库     | MySQL                            | 存储用户、线路、站点、车辆、历史客流等数据 |
| ORM        | SQLAlchemy                       | 操作数据库                                 |
| 数据处理   | Python + Pandas + NumPy          | 生成和处理模拟数据                         |
| 机器学习   | Scikit-learn / XGBoost           | 实现拥挤程度预测                           |
| 大模型接口 | DeepSeek / 通义千问 / OpenAI API | 实现 AI 出行助手                           |
| 接口通信   | Axios + RESTful API              | 前后端数据交互                             |
| 接口测试   | Apifox / Postman                 | 接口调试和文档管理                         |
| 项目管理   | Git / GitHub / Gitee             | 代码版本管理                               |
| 部署方式   | 本地部署 / 云服务器 + Nginx      | 项目演示和答辩部署                         |

```text
busmind-backend/
│

│

│   ├── bus_api.py                # 公交信息查询接口

│   ├── recommend_api.py          # 舒适推荐接口
│   └── ai_api.py                 # 大模型助手接口

├── models/                       # 数据库模型

│   ├── bus_line.py
│   ├── station.py
│   ├── vehicle.py

├── schemas/                      # 请求和响应数据结构

│   ├── predict_schema.py
```

## 十一、接口设计初稿

### 11.1 用户功能接口

| 接口               | 方法 | 说明         |
| ------------------ | ---- | ------------ |
| /api/user/register | POST | 用户注册     |
| /api/user/login    | POST | 用户登录     |
| /api/user/profile  | GET  | 获取用户信息 |
| /api/user/favorite | POST | 收藏路线     |
| /api/user/history  | GET  | 查询历史记录 |

------

### 11.2 公交信息查询接口

| 接口                              | 方法 | 说明             |
| --------------------------------- | ---- | ---------------- |
| /api/bus/lines                    | GET  | 查询所有公交线路 |
| /api/bus/lines/{line_id}          | GET  | 查询线路详情     |
| /api/bus/lines/{line_id}/stations | GET  | 查询线路站点     |
| /api/bus/vehicles                 | GET  | 查询车辆实时状态 |
| /api/bus/eta                      | GET  | 查询预计到站时间 |

------

### 11.3 智能预测与推荐接口

| /api/recommend/routes  | POST | 获取舒适路线推荐 |
| /api/recommend/explain | POST | 获取推荐理由解释 |
| /api/history/crowd     | GET  | 查询历史拥挤趋势 |
