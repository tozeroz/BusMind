# 二、当前项目定位

本项目聚焦乘客端公交舒适出行，不重点建设复杂管理员端。核心闭环是：

```text
查公交路线 -> 看 ETA -> 判断拥挤程度 -> 推荐舒适方案 -> AI 解释推荐原因
```
