## 当前协作边界

- 前端页面及组件由前端负责人实现，本轮不修改 `frontend/`。
- 后端升级必须兼容现有 `/api/v1/ai/travel` 请求。
- 实时路线、ETA、客载等使用业务服务，不把实时数据当作静态 RAG 文档。
- RAG 在业务工具链稳定后再接入。
